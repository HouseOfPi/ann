The 3 Paradigms

How Machines Learn

Machine learning algorithms are generally categorized by the way they are trained and the type of data they consume.

Supervised

The algorithm learns from a dataset where every example has a known label or "answer key". It learns the mapping from inputs to outputs.

Common Uses
Classification, Price Prediction

Unsupervised

The algorithm is given raw, unlabeled data. It must figure out the underlying structure, hidden patterns, or groupings on its own.

Common Uses
Customer Segmentation, Anomaly Detection

Reinforcement

An agent learns by interacting with an environment through trial and error. It receives rewards for good actions and penalties for bad ones.

Common Uses
Game AI, Robotics, Autonomous Vehicles
