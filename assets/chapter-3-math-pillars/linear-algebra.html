Mathematical Pillars · 01

Linear Algebra.

01
Vectors
A list of numbers representing a data point — an image, a word embedding, a set of features. Every input to a neural network is a vector.
input = [x₁, x₂, … xₙ]
02
Matrices
A table of numbers that encodes a transformation. The weights connecting two layers of a network live in a matrix.
W ∈ ℝ^(m×n)
03
Matrix Multiplication
The core operation of every forward pass. Multiplying the input vector by the weight matrix produces the next layer's activations — in one shot.
y = W · x
04
Dot Product
Measures alignment between two vectors. High dot product = strong match. Attention mechanisms in transformers are built entirely on dot products.
a · b = Σ aᵢbᵢ
W · x = y
